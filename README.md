# Kontrole
- ```W``` kamera unapred
- ```S``` kamera unazad
- ```A``` kamera ulevo
- ```D``` kamera udesno
- ```up``` exposure++
- ```down``` exposure--
- ```H``` hdr
- ```B``` bloom
- ```P``` blinn
- ```L``` spotlight 
- ```F``` spotlight u drugoj boji (dok je on)
- ```space``` treperenje spotlighta u drugoj boji (dok je on)
- ```esc``` izlaz
